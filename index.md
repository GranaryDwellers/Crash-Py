# Welcome to Crash-Py

**Python for Maths Teachers** — a practical crash course in Python designed for
secondary maths teachers who want to learn programming using modern tools on a
Mac.

Each lesson follows the same structure:

| Section | What you will find |
| --- | --- |
| **Introduction** | What the lesson covers and *why* it matters |
| **Do** | Guided, step-by-step practical activities |
| **Explore** | Open-ended questions and experiments to deepen understanding |

All lesson code examples are available to download so you can run them yourself.
